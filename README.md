# inventory-management-app

# 在庫管理アプリ

## 概要
家庭内の日用品や食品の在庫を管理するためのWebアプリです。
商品名、個数、価格などを登録し、一覧で確認できるようにしました。

## 作成目的
家にある物の買い忘れや重複購入を防ぐために作成しました。

## 使用技術
- Java
- Spring Boot
- HTML
- CSS
- Eclipse

## 主な機能
- 商品の登録
- 商品一覧表示
- 商品情報の編集
- 商品の削除

## 工夫した点
入力欄やボタンの配置を整理し、初めて使う人でも操作の流れが分かりやすい画面を意識しました。
また、商品名や個数を一覧で確認できるようにし、在庫状況を把握しやすくしました。
